## 🌐 Mission Overview

Welcome to **The Perimeter Lab**. This repository focuses on perfecting data structural layouts, web directory hierarchies, and user pathway architectures. 

Using the target mock grocery platform **"FreshDeals: Blueberries"**, this lab demonstrates how to implement non-functional, yet highly intuitive **Location** and **Attribute breadcrumbs** using semantic HTML5 and defensive CSS3. 

In secure full-stack design, user navigation isn't just about aesthetics—it minimizes user friction and maps data states transparently. By splitting breadcrumbs into structural paths vs. interactable attribute states, we reduce the app's cognitive load and secure user trust.

### 🔐 Lab Core Focus
* **Hierarchical Layouts:** Transforming linear document object lists into inline structural maps.
* **Semantic Differentiation:** Utilizing CSS pseudo-elements (`::after`) and targeted selectors to isolate mutable product attributes from static directory paths.
* **Defensive UI/UX:** Structuring breadcrumb indicators visually to protect user expectations of platform depth.
